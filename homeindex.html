Prazer, somos a Empodera-Tech

Por que a Empodera-Tech? Somos uma startup pequena de educação, mas com um potencial enorme para te ajudar na jornada do empoderamento digital. Confira abaixo alguns dos nossos cursos, e venha fazer parte do universo tech.

Curso de HTML e CSS
Nesse curso você irá aprender como criar uma página web e a estilizar.
Curso de raciocínio lógico
Nesse curso você irá desenvolver um pensamento computacional para criar soluções inovadoras.
JavaScript
Nesse curso você irá aprender como deixar as páginas em HTML interativas, com a linguagem de programação mais utilizada na web.
Curso de PHP
Nesse curso você irá aprender como coletar dados de formulários e deixar as páginas web interativas, e muito outros recursos.
